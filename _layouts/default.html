The problem

The problem was simple: I needed an application to monitor the time I invest in my personal projects and I wanted to do it myself.

The solution

The solution is ProTime, a minimal and easy to use application: play, pause and stop.
The interface is user friendly and attractive, I'm already working on some improvements and features that I felt the need while using the app.
Among these:

  • Pomodoro mode (starts a countdown activity)
  • Notification with controls
  • Graphs with statistics of time invested in projects
